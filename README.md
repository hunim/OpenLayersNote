# OpenLayers Note

## 文档介绍

这是一个关于OpenLayers学习全过程的笔记整理，是自己学习的过程记录，也整理平日里工作的一些积累，在此也分享给初学者，希望大家能一起进步，一起学习。为了满足广大初学者的需要，教程将坚持一个原则：即以**浅显易懂**为基础，以**有效实用**为最终目标的原则。

## 学习资料

* 目前最新版本 [v6.3.1](https://openlayers.org/download/)
